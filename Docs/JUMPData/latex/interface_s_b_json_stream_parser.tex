Parse some JSON. 

The JSON is assumed to be UTF8 encoded. This can be a full JSON document, or a part of one.

\begin{DoxyReturn}{Returns}
\begin{DoxyItemize}
\item SBJsonStreamParserComplete if a full document was found \item SBJsonStreamParserWaitingForData if a partial document was found and more data is required to complete it \item SBJsonStreamParserError if an error occured. (See the error property for details in this case.) \end{DoxyItemize}

Expect multiple documents separated by whitespace. 

If you set this property to true the parser will never return SBJsonStreamParserComplete. Once an object is completed it will expect another object to follow, separated only by whitespace.
