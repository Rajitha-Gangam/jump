+ (id) convertFromJavaUtilDateIfNeeded: (id)  anObject

This method convert an Java Util Date Dictionary to NSDate.

Usually used to decode this kind of date paserd from a JSON Object.

Check this post for more information: http://weblogs.asp.net/bleroy/archive/2008/01/18/dates-and-json.asp

Parameters:
anObjectAn object to try to convert.
Returns:
This method actually test if needs an conversion and converts if needed. If no convertion is needed will return the object passed as parameter with no changes.
