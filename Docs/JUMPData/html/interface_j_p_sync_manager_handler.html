Property Documentation

- (JPDBManager*) databaseManager [read, assign]

All Core Data operations of JPSyncManagerHandler is performed on a separated thread (background) in order to don't lock the Main Thread (or UI Thread).

Our handler uses a internal Database Manager (JPDBManager) to perform this operations. This property allows you to acess this Database Manager. You always should perform Database Operations using this Database Manager.

- (NSNumber *) currentProgress [read, assign]

Progress of current running task.

This property contain values from 0 to 100 indicating the percentage completed or nil if no task is running.
