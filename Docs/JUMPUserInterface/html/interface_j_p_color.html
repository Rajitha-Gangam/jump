Detailed Description

JPColor is an Color Object that encapsulate information about colors on different color spaces.

Is similar to the UIKit/UIColor class, but with a lot of more convenient and powerfull features. Contains several convenient methods to easily create colors and convert between spaces.

Creating colors

An example to how to create an JPColor with RGB values. 

 JPColor anColor = [JPColor initWithRed:255 G:0 B:0 opacity:100];

The above code create an red color. Note that JPColor use an Photoshop-like values (0-255) to represent RGB colors. So once created an JPColor is easy to convert it to an UIColor like this: 

 UIColor anUIColor = [[JPColor initWithRed:255 G:0 B:0 opacity:100] UIColor];

You also can create color using different color spaces, like CMYK; 

 JPColor *cmyk = [JPColor initWithCyan:0 M:100 Y:100: K:0 opacity:100];

Or an CSS string, like this: 

 JPColor *color = [JPColor initWithCSS:@"#2658e8"];

Colors Templates

JPColor includes an collection of color templates methods that follow the W3C CSS 3 Extended Colors Keywords. You can retrieve this methods simply calling his name, like this: 

 JPColor *seaGreenColor = [JPColor seagreen];

Colors Type Structures

JPColor return different structures types with values for different color spaces. See RGB, CMYK and HSV properties to learn more about it. Here one exampe retrieving an CMYK structure and displaying it: 

 JPColor *anColor = [JPColor greenColor];
 JPcmyk repr = [anColor CMYK];
 
 // Print values.
 NSLog( @"C:%i, M:%i, Y:%i, K:%i", repr.C, repr.M, repr.Y, repr.K );

+ (JPColor*) initWithCSS: (NSString *)  anCSSString

Initializes and returns a JPColor object using an String representing an CSS color.

This method understand several types of color representations usually encountered on CSS definitions as:

  • #fff
  • #FFFF
  • rgb(255,255,255)
  • rgba(100,100,100,50)

This value also can be one of the W3C CSS 3 Extended Colors Keywords.

Exceptions:
AnJPColorException if can't decode this string.
