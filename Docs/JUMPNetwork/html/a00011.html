Detailed Description

An Pipeline downstream Event that represent one "cancel" action.

You can cancel an downstream Event while he is processing sending an JPDefaultCancelEvent downstream as follow: 

 [pipeline sendDownstream:[JPDefaultCancelEvent init]];
Note:
Instead that The Pipeline are always an asynchronous I/O operation he process one Event at a time, that's why you can cancel an event while he is processing or waiting for some answer.
